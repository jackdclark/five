five.js

A library to overcomplicate 5.

Usage

Require the module
var five = require('five');
In the browser
<script type="text/javascript" src="./five.js"></script>
Basic 5
five(); // 5
Addition
five() + five(); // 10
Multiplication
five() * five(); // 25
Division
five() / five(); // 1
